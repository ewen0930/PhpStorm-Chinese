You can comment and uncomment lines and blocks of code using &shortcut:CommentByLineComment; and &shortcut:CommentByBlockComment;:

  • &shortcut:CommentByLineComment; comments or uncomments the current line or the selected block with single line comments (//...).
  • &shortcut:CommentByBlockComment; encloses the selected block in a block comment (/*...*/).

To uncomment a commented block, position the caret anywhere inside it and press &shortcut:CommentByBlockComment; .
